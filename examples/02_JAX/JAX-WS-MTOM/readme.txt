Usage:

1. Prepare:
mkdir ~/downloads
cp test.png ~/

2. In one console run:
java -cp JAX-WS-MTOM-1.0-SNAPSHOT.jar cz.vutbr.fit.knot.gja.ws.ImagePublisher

3. In second console run:
java -cp JAX-WS-MTOM-1.0-SNAPSHOT.jar cz.vutbr.fit.knot.gja.ws.ImageUpload
java -cp JAX-WS-MTOM-1.0-SNAPSHOT.jar cz.vutbr.fit.knot.gja.ws.ImageDownload
