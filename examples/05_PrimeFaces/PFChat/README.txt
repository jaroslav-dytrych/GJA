Not working in GlassFish!

Use WildFly https://www.wildfly.org/ (https://download.jboss.org/wildfly/21.0.0.Final/wildfly-21.0.0.Final.zip)
- unzip WildFly
- in NetBeans add WildFly server
- set Java for WildFly server to 1.8.x (else you will harm your installation)
- run it similar way as GlassFish
